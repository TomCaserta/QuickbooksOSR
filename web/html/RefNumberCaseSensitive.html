RefNumberCaseSensitive

A string of characters that refers to this transaction and that can be arbitrarily changed by the QuickBooks user.

Prior to SDK 5.0, this tag was used rather than the older RefNumber tag, because it provides much better performance in certain circumstances, such as if the refNumber value contains letters, not just digits. As of SDK 5.0, both the RefNumber tag and this tag yield the same good performance in all these scenarios. However, you can still use this tag if you want to do a case-sensitive search, because RefNumber does a case-insensitive search.
