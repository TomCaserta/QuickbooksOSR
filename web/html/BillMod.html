BillMod

BillMod allows you to edit an existing Bill, similar to editing a Bill in the Enter Bills form in the QuickBooks UI.
Notice that you cannot link a PurchaseOrder to an existing Bill using the BillMod request. You can link to a PurchaseOrder only in new Bills, using BillAdd.
Notice also that some fields in a BillMod request cannot be cleared. If any of the following fields is included in a bill modify request, it must contain a value:
  • VendorRef
  • APAccountRef
  • TxnDate
  • DueDate
  • ClearExpenseLines
  • ClearItemLines
Within ExpenseLineMod:
  • Amount
  • BillableStatus
Within ItemLineMod or ItemGroupMod:
  • ItemRef
  • Quantity
  • Cost
  • Amount
  • BillableStatus
  • OverrideItemAccountRef
(For more details about what it means for a field to be clearable or not clearable, see the QB SDK Programmer's Guide.)
