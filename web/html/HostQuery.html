A HostQueryRq request message returns information about the exact QuickBooks product and version with which you are communicating. Note: If you include an explicit end tag in a HostQueryRq request, you will receive an error if there is any space between the opening tag and the end tag. The following table shows valid and invalid ways of constructing this request:
