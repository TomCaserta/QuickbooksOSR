STRTYPE
Strings are alphanumeric characters. The maximum allowed field length of each STRTYPE element is shown in the Max column of the table above.

A STRTYPE can include any standard, predefined XML entity reference. The following table shows the XML entity references for a few of the most common special characters:

To show this character: Use this entity reference:
< (less than) &lt;
> (greater than) &gt;
' (single quote) &apos;
" (double quote) &quot;
& (ampersand) &amp;

Note: Three special characters will pass validation even if they are not encoded: the greater-than sign (>), the single-quote character (') and the double-quote character (").
