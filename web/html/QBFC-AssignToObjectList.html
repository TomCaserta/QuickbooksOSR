AssignToObjectList
A list of AssignToObject enum values. If this is a private data extension definition (the OwnerID value is a non-zero GUID), then AssignToObject is optional and any of the enumerated objects listed below can be specified for AssignToObject.

If this is a custom field definition (the OwnerID value is 0), then AssignToObject is required.
If this data ext definition is for a custom field, then only Customer, Employee, Vendor, or Item can be specified in AssignToObject. Notice that assigning to Customer or Item results in automatic propagation (assigning) to the following objects, where the custom field is available at the transaction level or transaction line item level:
  • CreditMemo
  • Estimate
  • Invoice
  • PurchaseOrder (only if AssigntoObject is set to Item)
  • SalesOrder
  • SalesReceipt
