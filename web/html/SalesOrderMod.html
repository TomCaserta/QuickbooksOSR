Some fields in a SalesOrderMod request cannot be cleared. If any of the following fields is included in a sales-order modify request, it must contain a value:
  • CustomerRef
  • TemplateRef
  • TxnDate
  • DueDate
  • ShipDate
  • ItemSalesTaxRef
  • IsManuallyClosed
  • IsToBePrinted
Within SalesOrderLineMod or SalesOrderLineGroupMod:
  • ItemRef
  • Quantity
  • Rate
  • Amount
  • SalesTaxCodeRef
  • IsManuallyClosed
(For more details about what it means for a field to be clearable or not clearable, see the QuickBooks SDK Programmer's Guide.)
