ChargeMod

Some fields in a ChargeMod request cannot be cleared. If any of the following fields is included in a charge modify request, it must contain a value:
  • CustomerRef
  • TxnDate
  • ItemRef
  • Quantity
  • Rate
  • Amount
  • ARAccountRef
  • OverrideItemAccountRef
(For more details about what it means for a field to be clearable or not clearable, see the QB SDK Programmer's Guide.)
