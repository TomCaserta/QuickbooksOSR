ApplyCheckToTxn
If the check is being written to refund for some transaction, you need to supply the transaction ID and amount.

Allows you to specify a transaction to be paid by this check.
Any transactions linked from a check in the ApplyToTxnAdd aggregate MUST:
a.Use the same AR account specified in the ExpenseLineAdd/AccountRef which MUST be an ARAccount.
b. Apply to the same Entity referred to in the Check’s PayeeEntityRef
