CompanyActivity
Returns the date and time when the company file was last restored and when it was last condensed. Returns the default date of 1970-01-01 (UTC) for these elements if a restore or condense hasn't happened for this company file. (For example, 1970-01-01 will be represented as 1969-12-31T16:00:00-08:00 in a time zone that is eight hours behind UTC.)
