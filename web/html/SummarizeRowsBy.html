SummarizeRowsBy
SummarizeRowsBy (along with IncludeColumn, in most cases) determines what data is calculated for this report and controls how the rows are organized and labeled. For example, if you set the value to Account, the report's row labels might be "Checking," "Savings," and so on.

Values:
Account, BalanceSheet, Class, Customer, CustomerType, Day, Employee, FourWeek, HalfMonth, IncomeStatement, ItemDetail, ItemType, Month, Payee, PaymentMethod, PayrollItemDetail, PayrollYtdDetail, Quarter, SalesRep, SalesTaxCode, ShipMethod, TaxLine, Terms, TotalOnly, TwoWeek, Vendor, VendorType, Week, Year

No default value.
