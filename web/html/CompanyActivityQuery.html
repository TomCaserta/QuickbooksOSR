A CompanyActivityQueryRq request message returns LastRestoreTime (the time when the company file was last restored from a back up), and LastCondenseTime (the time the company file was last condensed). It returns the default date of 1970-01-01 (UTC) for these elements if a restore or condense hasn't happened for this company file. (For example, 1970-01-01 will be represented as 1969-12-31T16:00:00-08:00 in a time zone that is eight hours behind UTC.) Note: If you include an explicit end tag in a CompanyActivityQuery request, you will receive an error if there is any space between the opening tag and the end tag.
