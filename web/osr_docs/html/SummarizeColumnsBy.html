SummarizeColumnsBy
SummarizeColumnsBy determines which data the report calculates and how the columns will be labeled across the top of the report. For example, if you set the value to TwoWeek, the report's column labels might be "Feb 10-23, 02," "Feb 24-Mar9, 02," and so on.

Values:
Account, BalanceSheet, Class, Customer, CustomerType, Day, Employee, FourWeek, HalfMonth, IncomeStatement, ItemDetail, ItemType, Month, Payee, PaymentMethod, PayrollItemDetail, PayrollYtdDetail, Quarter, SalesRep, SalesTaxCode, ShipMethod, Terms, TotalOnly, TwoWeek, Vendor, VendorType, Week, Year

No default value.
