Quantity

  • For transactions: If an item line add on a transaction request specifies Quantity and Amount but not Rate, QuickBooks will use Quantity and Amount to calculate Rate. Likewise, if a request specifies Quantity and Rate but not Amount, QuickBooks will calculate the Amount. If a transaction add request includes a reference to an ItemDiscount item, do not include a Quantity element as well, or you will get an error.

  • For Item requests: Quantity indicates how many of this item there are.
