LinkToTxnIDList

The LinkToTxnID aggregate allows to link an entire transaction. If you want to link individual lines in a transaction, use the LinkToTxn aggregate.

You can use both LinkToTxnID and LinkToTxn aggregates in the same request, as long as these aggregates do NOT link to the same TxnID. If both aggregates are used and do link to the same TxnID, you receive an error 3000.

Also, if you attempt to invoke LinkToTxnID with the same TxnID, more than once, you get an error 3177. If the transaction specified by LinkToTxnID is already closed or fully received and the ItemReceiptAdd request does not contain any other lines, you get error 3180 for attempting to save an empty transaction. You get a warning if you attempt to link to a transaction that's already closed.
