• EmployeeStateTaxesDetail lists state taxes withheld for employees
  • PayrollItemDetail lists the payroll transactions on which each payroll item appears
  • PayrollReviewDetail gives detailed information about how QuickBooks calculates tax amounts on employee paychecks and in year-to-date transactions
  • PayrollTransactionDetail shows the line-item detail that appears on each payroll transaction
  • PayrollTransactionsByPayee lists payroll transactions, grouping them by payee

Values:
EmployeeStateTaxesDetail, PayrollItemDetail, PayrollReviewDetail, PayrollTransactionDetail, PayrollTransactionsByPayee

No default value.
