DeliveryPolicy
Determines whether or not an application should be launched to receive events:
  • dpDeliverAlways--QuickBooks will always deliver events to the application, starting it through COM if necessary. The application's COM pointer is released by QuickBooks when the company file closes.
  • dpDeliverOnlyIfRunning--QuickBooks will deliver events to an application only when it is already running. The application's COM pointer is released by QuickBooks when the company file closes.
