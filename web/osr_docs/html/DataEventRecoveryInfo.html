DataEventRecoveryInfo
You can use the DataEventRecoveryInfoQuery request to find out whether data event recovery information exists for the given subscriber. If it does, then data events have been lost and need to be recovered. (Note that there will be no data event recovery information to return if TrackLostEvents was "tleNone" in the subscriber's DataEventSubscriptionAdd request.)

Returned data event recovery information currently consists only of DataEventRecoveryTime. For more details about the event-notification feature, see the chapter called "Event Notification" in the QB SDK Programmer's Guide.
