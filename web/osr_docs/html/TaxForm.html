TaxForm
The tax form that the QuickBooks user expects to file for this company's taxes. If a TaxForm value is assigned (that is, if it is not tfOtherOrNone), the QuickBooks user can associate each account with a tax form line. (An account's tax form line information will show up in the response from an account query.)

Values:
Form1040, Form1065, Form1120, Form1120S, Form990, Form990PF, Form990T, OtherOrNone

No default value.
