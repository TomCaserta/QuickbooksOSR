Some fields in a CreditMemoMod request cannot be cleared. If any of the following fields is included in a credit-memo modify request, it must contain a value:
  • CustomerRef
  • ARAccountRef
  • TemplateRef
  • TxnDate
  • IsPending
  • DueDate
  • ShipDate
  • ItemSalesTaxRef
  • IsToBePrinted
Within CreditMemoLineMod or CreditMemoLineGroupMod:
  • ItemRef
  • Quantity
  • Rate
  • Amount
  • SalesTaxCodeRef
  • OverrideItemAccountRef
(For more details about what it means for a field to be clearable or not clearable, see the QB SDK Programmer's Guide.)
