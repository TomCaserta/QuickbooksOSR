Some fields in an EstimateMod request cannot be cleared. If any of the following fields is included in an estimate modify request, it must contain a value:
  • CustomerRef
  • TemplateRef
  • TxnDate
  • IsActive
  • CreateChangeOrder
  • DueDate
  • ItemSalesTaxRef
Within EstimateLineMod or EstimateLineGroupMod:
  • ItemRef
  • Quantity
  • Rate
  • Amount
  • SalesTaxCodeRef
  • Markup
(For more details about what it means for a field to be clearable or not clearable, see the QuickBooks SDK Programmer's Guide.)
