GUIDTYPE
A GUIDTYPE value can be zero (0), or it can take the form
{XXXXXXXX-XXXX-XXXX-XXXX-XXXXXXXXXXXX}
where X is a hexadecimal digit. For example:
{6B063959-81B0-4622-85D6-F548C8CCB517}

GUIDTYPE strings are globally unique. To generate them, you can use GUIDGEN.EXE, a tool that comes with Microsoft Visual Studio. GUIDTYPE values are not case-sensitive, so if you enter a value with lower-case letters, it will be saved and returned with upper-case letters.

The requestID in the qbXML message is a UUIDTYPE, which is a GUIDTYPE, but one that QuickBooks treats in a special way, such as relaxing the format requirements in the RequestID.
